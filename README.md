# Microservice

### Constructed as parte of an IgniteLab from Rocketseat

This is a microservice created to learn some core concepts of microservices and some design patterns such as:

- Repository pattern
- Factory pattern

It is powered by those technologies:

- Prisma ORM
- Nestjs
- Jest

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
